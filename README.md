# Simple Geometry Visualizer
Simple c++ app that can bee used for 2D visualizations of points and lines using SDL2 library for graphics. Uses some geometry algorythms, moving and zooming of view.
## Usage
It is a visual studio solution, requires linking SDL2 library.
### Controls
arrows - movement,
q - zoom in,
w - zoom out,
a - resests zoom to 1.
